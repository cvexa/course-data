4.1  Направете SQL заявка за  информация за филмите и имената на техните продуценти : 
SELECT * FROM `films` JOIN `producers` ON `films`.`producer_id` = `producers`.`producer_id` WHERE `films`.`date_deleted` is NULL 






4.2 Направете SQL заявка за най-младия продуцент.
SELECT max(`birth_year`) FROM `producers` WHERE `date_deleted` is null 


4.3 Направете SQL заявка за средното количество билети, продадени за филмите в БД;
SELECT avg(`tickets_sold`) FROM `films` WHERE `date_deleted` is null 

4.4 Направете SQL заявка за филмите, съдържащи ‘ger ’ в името си.
SELECT * FROM `films` WHERE `film_name` like '%ger%' and `date_deleted` is null
